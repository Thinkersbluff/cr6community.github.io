# What Makes the Display Hardware Important to the CR6Community Firmware?

The CR6Community Firmware is specifically optimized for monitoring and controlling the Marlin or Klipper firmware on the motherboard.  It ONLY runs correctly on DWIN touchscreen devices with 272 x 480 pixel displays, and which run the DGUS2 operating system on a T5L ASIC.  

If a user decides that they no longer value the DWIN display device, they no longer need to install or run the CR6Community Firmware.

Conversely, if the user selects display hardware which is NOT one of the devices described below, they likely can not run the CR6Community Firmware, unless they "port" the screens to the  screen resolution of their chosen display hardware.
