# Local Preview Quick Start

Use these commands when you come back to this project.

## 1) Start or restart the Jekyll server

```powershell
cd B:\3D_Objects\GitHubClones\cr6community.github.io
bundle exec jekyll serve --livereload
```

## 2) Open the local site in your browser

- http://127.0.0.1:4000/cr6community.github.io/

## If changes do not appear right away

- Refresh the browser page.
- Use Ctrl+F5 for a hard refresh.
